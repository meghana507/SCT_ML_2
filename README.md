# Retail Store Customer Segmentation

## Project Overview
This project uses **Unsupervised Learning** to group retail customers into distinct segments. By identifying patterns in purchase history, businesses can create targeted marketing strategies and improve customer retention.

## Dataset
The project utilizes the **Mall Customer Dataset**, which includes:
* Customer IDs
* Annual Income
* Spending Score (assigned based on purchase behavior)

## Algorithm
* **K-Means Clustering:** Used to partition customers into groups with similar characteristics.
* **Elbow Method:** Employed to determine the optimal number of clusters ($K$).

## Key Objectives
* Identify high-value customer segments.
* Visualize clusters to understand spending patterns.
* Provide actionable insights for retail marketing teams.
